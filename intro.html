Intro about HTML

html stands for Hyper text mark-up language which is used for creating web pages. HTML consists of elements and it describes the structure of web page.

Format of html file

- root of an page - it contains meta info.(data about data) - its the title of the page which is visible in browser tab <body> - its a document body, in this all elements are visible in web page <h1> - it define large heading <p> - define paragraph </p> </body> </html>
