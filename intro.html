Intro about HTML

html stands for Hyper text mark-up language which is used for creating web pages. HTML consists of elements and it describes the structure of web page.We should save the html files with . htm or .html extensions.

Format of html file

html - root of an page
head - it contains meta info.(data about data)
title - its the title of the page which is visible in browser tab
body - its a document body, in this all elements are visible in web page
h1 - it define large heading
p - define paragraph
